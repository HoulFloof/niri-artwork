# niri artwork
![works with niri](https://github.com/niri-wm/artwork/raw/refs/heads/main/official-branding/workswithniri.svg)
---
A curated collection of artwork (wallpapers, alternative user-created logos, official branding, and badges) for the [niri](https://github.com/niri-wm/niri/) Wayland compositor created by [YaLTeR](https://github.com/YaLTeR/). 

# Rules for submission
1. You must be the creator of all submitted artwork.
2. You must not create artwork using Generative AI. Minor usage, including generative fill, background removal, healing, etc. are allowed, but any artwork with a significant AI contribution will not be accepted.
3. You must keep your audience in mind. Please do not submit artwork that sexualizes or stereotypes people.

# Maintainers
This repository is maintained by the [@niri-wm/branding](https://github.com/orgs/niri-wm/teams/branding) team.
